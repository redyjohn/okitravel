# 🌴 2026 沖繩親子自駕旅行

## 基本資訊
- 日期：2026/06/27 – 06/30
- 人數：11大人 + 4嬰兒
- 交通：4台自駕
- 風格：購物 + 美食 + 親子慢旅

---

## 🚗 核心策略

- 每天 2～4 個核心點
- 減少集合次數（4台車）
- 每天至少 1 間地陪推薦餐廳
- 美國村安排夕陽＋夜景
- 最後一天：國際通 → iias → 機場（順向不回頭）

---
